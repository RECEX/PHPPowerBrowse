PHP Power Browse v1.2
----------------------

Information:PHP Power Browse is a complex file browsing script which allows the user to see basic file stats and view souce of allows file types. It also includes simple configurable security on specified files and directory, as well as customizable colors and feel. All images are built into the script and only one file is needed!
Written by Devin Smith (php@arzdev.com)

Installation:
1. Copy the 'browse.php' to your web server.
2. Modify the config in the top of 'browse.php' to meet your needs.
3. Run the script with 'http://server.com/browse.php'
